# sc-qs-dc-php

ScorpioCoing QuickStart Docker Nginx Php stack

## Basic Usage

- clone git
- open terminal and go into \_dockerfiles
- run docker command to start
  ```
  docker compose up -d
  ```
- run docker command to stop
  ```
  docker compose down
  ```
- http://localhost:6080
